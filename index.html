Die Karte

So kann ihre fertige Fotokarte aussehen: Das Foto kommt auf einer Karte mit einem farblich passenden Kuvert sowie einem weissen Innenblatt - alles verpackt in einem 2. Kuvert. Eine Karte kostet 3.50 Franken. Dazu kommen noch die Versandskosten.

Die Bestellung

1. Schauen sie sich die Fotos an

Scrollen sie durch die Gallerie und Wählen sie ein Foto aus.

2.Suchen sie sich ein Spruch aus.

Wollen sie ein Spruch auf ihrer Karte? Suchen sie einen von den Vorgefertigten aus, oder erfinden sie ihren eigenen.

3. Wählen sie das Kartenformat

Wählen sie zwischen verschiedenen Kartengrössen aus, um eine perfekte Karte zu gestalten.

4. Bestellen sie ihre Karte

Bestellen sie ihre Karte per E-mail. Geben sie alle wichtigen Infos an. (Foto, Spruch, Kartengrösse und Lieferadresse.) E-Mail: sebastiansgrusskarten@mail.ch

5. Die Karte wird erstellt

Ich erstelle nun ihre persönliche Karte.

6. Versand

Die Karte wird nun an Ihre Adresse versandt und sie können sie einsetzten.
